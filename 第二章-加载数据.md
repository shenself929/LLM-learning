## 加载非结构化数据

- 问题：加载非结构化数据（如文本或图像）。

```python
import requests   进口的请求

# URL to download the txt file from
txt_url = "https://machine-learning-python-cookbook.s3.amazonaws.com/text.txt"

# 获取 txt 文件

#将其写入本地的 text.txt 文件

    f.write(r.content)

    text=f.read()

print(text)
```
